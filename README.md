Gestion des employés — Mini-app

Une petite application front-end pour afficher, filtrer et rechercher des employés.

## Aperçu
- Interface simple (HTML + JavaScript) utilisant Bootstrap pour le style.
- Permet d'afficher la liste des employés, calculer le total des salaires, filtrer par département et spécialisation, et rechercher par ID.

## Fonctionnalités
- Afficher tous les employés
- Rechercher un employé par ID
- Filtrer par département → spécialisation (listes dépendantes)
- Calculer la somme des salaires
- Alertes utilisateur pour les actions

## Fichiers principaux
- `index.html` — page principale et interface utilisateur
- `employee_details.js` — logique JavaScript (données et fonctions)
- `style.css` — styles personnalisés et thèmes

## Exécution / Test rapide
1. Ouvrir `index.html` directement dans un navigateur (double-clic) pour un usage simple.


## Développement
- Modifier `employee_details.js` pour ajouter ou éditer des employés (tableau `employees`).
- Les fonctions utilitaires sont commentées pour faciliter les adaptations.

## Suggestions d'améliorations
